# Zoom Limit 6 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 将区域裁剪的手动与随机缩放倍率统一调整为 1.5～6.0。

**Architecture:** 在裁剪模块公开缩放倍率边界常量，并让随机裁剪、有效性校验、生成器钳制和控制面板滑杆共同引用。现有裁剪覆盖率、可见比例、尝试次数和回退逻辑保持不变。

**Tech Stack:** TypeScript 7、React 19、Vitest 4、Testing Library

## Global Constraints
- 缩放倍率最小值保持 1.5。
- 缩放倍率最大值调整为 6.0。
- 随机裁剪最多尝试 20 次。
- 内容覆盖率至少为 15%，源图可见比例最多为 70%。
- 随机失败时保留现有 2 倍居中回退。
